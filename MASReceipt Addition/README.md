## MASReceipt Addition

任意のアプリケーションに `_MASReceipt` を付加します。<br>
このファイルは,Mac App Storeから入手したアプリケーションのパッケージに必ず付加されているファイルです。<br>
Mac App Store以外から入手したアプリケーションにもこれを付加することで,Launchpadからアプリケーションを削除できるなど,Mac App Storeのアプリケーションと同等の機能が追加されます。

### 使い方

```sh
addMASRecepit someApplication.app
```
この操作により,アプリケーション `someApplication` に _MASReceipt を付加します。
